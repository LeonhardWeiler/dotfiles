# Keybindings

Everything bound on this system, per program. Generated by reading the tracked
configs ‚Äî it is a snapshot, not a second source of truth: the authority is
always the config file named in each section.

Note on the keyboard: dwl loads **us / colemak_dh** with `caps:swapescape`
(Caps Lock is Escape and vice versa). All letter keys below are named the way
they are **labelled on a Colemak-DH layout**, which is how they are written in
`config.h`.

## dwl (compositor) ‚Äî `config/dwl/config.h`

`MOD` = **Alt**. Compile-time config: changing anything here needs
`./install --dwl` plus a new session.

### Launch / windows

| Keys                | Action                                                     |
| ------------------- | ---------------------------------------------------------- |
| `MOD+Shift+Return`  | Terminal (foot)                                            |
| `MOD+Shift+P`       | App launcher (`rofi -show drun`)                           |
| `MOD+Shift+T`       | File browser (`rofi -show filebrowser`)                    |
| `MOD+Shift+N`       | Browser (zen-browser)                                      |
| `MOD+Shift+C`       | Close the focused window                                   |
| `MOD+Shift+Q`       | Quit dwl ‚Äî asks for confirmation in rofi first             |
| `MOD+L`             | Lock the screen (waylock)                                  |
| `MOD+V`             | Toggle floating for the focused window                     |
| `MOD+M`             | Toggle fullscreen                                          |

### Layout / focus

| Keys            | Action                                              |
| --------------- | --------------------------------------------------- |
| `MOD+J`         | Focus next window in the stack                      |
| `MOD+K`         | Focus previous window in the stack                  |
| `MOD+Return`    | Zoom ‚Äî swap the focused window into the master area |
| `MOD+Shift+H`   | Shrink the master area (‚àí2 %)                       |
| `MOD+Shift+L`   | Grow the master area (+2 %)                         |
| `MOD+Shift+I`   | One more window in the master area                  |
| `MOD+Shift+O`   | One fewer window in the master area                 |

New windows attach at the **bottom** of the stack (`attachbottom.patch`), so
opening one never displaces what you are looking at.

### Tags (9)

| Keys            | Action                                  |
| --------------- | --------------------------------------- |
| `MOD+1` ‚Ä¶ `9`   | View tag 1‚Äì9                            |
| `MOD+Shift+1` ‚Ä¶ `9` | Move the focused window to tag 1‚Äì9  |

### Media / hardware

| Keys       | Action                                       |
| ---------- | -------------------------------------------- |
| `MOD+F1`   | Mute toggle (`vol_ctl mute`, OSD via wob)    |
| `MOD+F2`   | Volume ‚àí5 % (`vol_ctl down`)                 |
| `MOD+F3`   | Volume +5 % (`vol_ctl up`)                   |
| `MOD+F5`   | Brightness down (`bright_ctl down`)          |
| `MOD+F6`   | Brightness up (`bright_ctl up`)              |
| `Print`    | Select a region ‚Üí screenshot to the clipboard (`grim`+`slurp`+`wl-copy`) |

### Mouse (with `MOD` held)

| Button   | Action           |
| -------- | ---------------- |
| Left     | Move the window  |
| Middle   | Toggle floating  |
| Right    | Resize the window|

### Escape hatches

| Keys                 | Action                        |
| -------------------- | ----------------------------- |
| `Ctrl+Alt+F1` ‚Ä¶ `F12`| Switch to virtual terminal 1‚Äì12 |

`Ctrl+Alt+Backspace` is deliberately **not** bound.

## Neovim ‚Äî `config/nvim/`

Leader is `Space`. `Space`, `Backspace` and `F1` are mapped to nothing in normal
mode on purpose, so a stray press does not do something surprising.

### Own mappings ‚Äî `lua/config/keymaps.lua`

| Keys              | Mode   | Action                                              |
| ----------------- | ------ | --------------------------------------------------- |
| `<leader>pv`      | n      | Open Oil (file browser)                             |
| `J` / `K`         | visual | Move the selection down / up, reindented            |
| `J`               | n      | Join lines, keeping the cursor where it was         |
| `<C-d>` / `<C-u>` | n      | Half page down / up, recentred                      |
| `<C-f>` / `<C-b>` | n      | Full page down / up, recentred                      |
| `n` / `N`         | n      | Next / previous search hit, recentred and unfolded  |
| `<leader>r`       | x      | Replace the selection with the yank register (no clobber) |
| `<leader>S`       | n      | Search & replace the word under the cursor          |
| `<leader>sd`      | n      | Put diagnostics into the quickfix list              |
| `<leader>ob`      | n      | Open the link under the cursor in the browser       |
| `j` / `k`         | n      | Move by *display* line ‚Äî markdown/text buffers only |

### Oil (inside the file browser) ‚Äî `lua/plugins/oil.lua`

| Keys      | Action                              |
| --------- | ----------------------------------- |
| `g?`      | Show Oil's own help                 |
| `<CR>`    | Open the entry                      |
| `<C-s>`   | Open in a vertical split            |
| `<C-h>`   | Open in a horizontal split          |
| `<C-t>`   | Open in a new tab                   |
| `<C-p>`   | Preview                             |
| `<C-c>`   | Close Oil                           |
| `<C-l>`   | Refresh                             |
| `-`       | Up to the parent directory          |
| `_`       | Jump to the current working directory |
| `` ` ``   | `cd` into the directory             |
| `~`       | `cd` into it, tab-scoped            |
| `gs`      | Change the sort order               |
| `gx`      | Open with the external default app  |

Completion (blink.cmp) runs the plugin's **default** preset: `<C-space>` opens
the menu, `<C-n>`/`<C-p>` cycle, `<CR>` accepts, `<Tab>`/`<S-Tab>` jump through
snippet placeholders.

## Programs running on their defaults

These carry **no** custom bindings in this repo ‚Äî listed so it is clear that
nothing was changed, with the handful of keys worth remembering:

| Program       | Config              | Notable defaults                                                                 |
| ------------- | ------------------- | -------------------------------------------------------------------------------- |
| foot          | `config/foot/foot.ini` | `Ctrl+Shift+C`/`V` copy & paste, `Ctrl+Shift+R` search, `Ctrl+Shift+N` new window, `Ctrl++`/`-` font size |
| rofi          | `config/rofi/`      | Type to filter, `‚Üë`/`‚Üì` or `Ctrl+N`/`Ctrl+P` to move, `Enter` to run, `Esc` to cancel, `Shift+Enter` runs in a terminal |
| btop          | `config/btop/btop.conf` | `Esc`/`m` menu, `q` quit, `+`/`-` update interval, `f` filter, `k` kill process |
| waylock       | none (CLI flags in `config.h`) | Type the password, `Enter` to unlock ‚Äî no other input      |
| KeePassXC     | `config/keepassxc/` | `Ctrl+C` copy password, `Ctrl+B` copy username, `Ctrl+F` search, `Ctrl+L` lock database |
